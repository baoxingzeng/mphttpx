# A polyfill for mini-program.

## install
```bash
npm install mphttpx
```

## usage
```javascript
import {
    TextEncoder,
    TextDecoder,

    Event,
    EventTarget,
    CustomEvent,
    ProgressEvent,

    Blob,
    File,
    FileReader,

    FormData,
    URLSearchParams,

    AbortSignal,
    AbortController,

    XMLHttpRequest,
    fetch,
    Headers,
    Request,
    Response
} from "mphttpx";
```
